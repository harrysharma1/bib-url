## bibcli booklet

Generate @booklet bibtex entry

### Synopsis

This will generate an @booklet bibtex entry e.g.
	
@booklet{CitekeyBooklet,
	title        = "<Title>",
	author       = "<Lastname, Firstname>", 
	howpublished = "<How Published>",
	address 	 = "<Address>",
	year         = <2002>,
	editor		 = "<Lastname, Firstname>",
	volume		 = "<50>",
	number		 = "<1>",
	series		 = "<Series>",
	organisation = "<Organisation>",
	month		 = <jul>,
	note		 = "<Note>"
}

Required:
	- title
	- author
	- howpublished
	- address
	- year
Optional:
	- editor
	- volume
	- number
	- series
	- organisation
	- month
	- note
<>: indicates that it is a example value and should be changed.
