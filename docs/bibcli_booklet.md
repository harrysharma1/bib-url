## bibcli booklet

Generate @booklet bibtex entry

### Synopsis

This will generate an @booklet bibtex entry e.g.
	
	@booklet{CitekeyBooklet,
 		title        = "Canoe tours in {S}weden",
  		author       = "Maria Swetla", 
  		howpublished = "Distributed at the Stockholm Tourist Office",
  		month        = jul,
  		year         = 2015
	}
