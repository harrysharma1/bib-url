## bibcli proceedings

Generate @proceedings bibtex entry

### Synopsis

This will generate an @proceedings entry e.g.
	
@proceedings{CitekeyProceedings,
	title     = "<Title>",
	year	  = <2002>,
	editor	  = "<Lastname, Firstname>",
	volume    = "<1>",
	number	  = "<50>",
	series	  = "<Series>",
	address   = "<Address>",
	month	  = <jul>,
	publisher = "<Publisher>"
}

Required:
	- title
	- year
Optional:
	- editor
	- volume
	- number
	- series
	- address
	- month
	- publisher
<>: indicates that it is a example value and should be changed.
