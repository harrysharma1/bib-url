## bibcli phdthesis

Generate @phdthesis bibtex entry

### Synopsis

This will generate an @phdthesis bibtex entry e.g.
	
@phdthesis{CitekeyPhdthesis,
	author  = "<Lastname, Firstname>",
	title   = "<Title>",
	school  = "<School>",
	year    = <2002>,
	type	= "<Type>",
	address = "<Address>",
	month   = <jul>,
	note	= "<Note>"
}

Required:
	- author
	- title
	- school
	- year
Optional

	- address
	- month
	- note
<>: indicates that it is a example value and should be changed.
