## bibcli mastersthesis

Generate @mastersthesis bibtex entry

### Synopsis

This will generate an @mastersthesis bibtex entry e.g.
	
@mastersthesis{CitekeyMastersthesis,
	author  = "<Lastname, Firstname>",
	title   = "<Title>",
	school  = "<School>",
	year    = <2002>,
	type	= "<Type>",
	address = "<Address>",
	month   = <jul>,
	note	= "<Note>"
}

Required:
	- author
	- title
	- school
	- year
Optional:
	- type
	- address
	- month
	- note
<>: indicates that it is a example value and should be changed.
