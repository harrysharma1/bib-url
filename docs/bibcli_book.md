## bibcli book

Generate @book bibtex entry

### Synopsis

This will generate an @book bibtex entry e.g.
	
@book{CitekeyBook,
	author    = "<Lastname, Firstname>",
	title     = "<Title>",
	publisher = "<Publisher>",
	address   = "<Address>",
	year      = <2002>
}

Required:
	- author
	- title
	- publisher
	- address
	- year
<>: indicates that it is a example value and should be changed.
