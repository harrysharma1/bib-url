## bibcli book

Generate @book bibtex entry

### Synopsis

This will generate an @book bibtex entry e.g.
	
	@book{CitekeyBook,
  		author    = "Leonard Susskind and George Hrabovsky",
  		title     = "Classical mechanics: the theoretical minimum",
  		publisher = "Penguin Random House",
  		address   = "New York, NY",
  		year      = 2014
	}
