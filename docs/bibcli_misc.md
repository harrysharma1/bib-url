## bibcli misc

Generate @misc bibtex entry

### Synopsis

This will generate an @misc bibtex entry e.g.
	
@misc{CitekeyMisc,
	title        = "<Title>",
	author       = "<Lastname, Firstname>",
	howpublished = "<How published>",
	year         = <2002>,
	note         = "<Note>"
}

Required:
	- title
	- author
	- howpublished
	- year
	- note
Optional:
	- !*
<>: indicates that it is a example value and should be changed.
