## bibcli misc

Generate @misc bibtex entry

### Synopsis

This will generate an @misc bibtex entry e.g.
	
	@misc{CitekeyMisc,
  		title        = "Pluto: The 'Other' Red Planet",
  		author       = "{NASA}",
  		howpublished = "\url{https://www.nasa.gov/nh/pluto-the-other-red-planet}",
  		year         = 2015,
  		note         = "Accessed: 2018-12-06"
	}
