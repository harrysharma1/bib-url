## bibcli inbook

Generate @inbook bibtex entry

### Synopsis

This will generate an @inbook bibtex entry e.g.
	
@inbook{CitekeyInbook,
	author    = "<Lastname, Firstname>",
	title     = "<Title>",
	booktitle = "<Book Title>",
	publisher = "<Publisher>",
	year 	  = <2002>,
	editor	  = "<Lastname, Firstname>",
	volume	  = "<50>",
	number	  = "<1>",
	series	  = "<Series>",
	address   = "<Address>",
	edition	  = "<2nd>",
	month	  = <jul>,
	pages     = "<10--100>",
	note	  = "<Note>",
}

Required:
	- author
	- title
	- booktitle
	- publisher
	- year
Optional:
	- editor
	- volume
	- number
	- series
	- address
	- edition
	- month
	- pages
	- note
<>: indicates that it is a example value and should be changed.
