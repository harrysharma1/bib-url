## bibcli manual

Generate @manual bibtex entry

### Synopsis

This will generate an @manual bibtex entry e.g.
	
@manual{CitekeyManual,
	title        = "<Title>",
	year		 = <2002>,
	author       = "<Lastname, Firstname>",
	organisation = "<Organisation>",
	address      = "<Address>",
	edition		 = "<2nd>",
	month		 = <jul>,
	note		 = "<Note>"
}

Required:
	- title
	- year
Optional:
	- author
	- organisation
	- address
	- edition
	- month
	- note
<>: indicates that it is a example value and should be changed.
