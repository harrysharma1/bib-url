## bibcli article

Generate @article bibtex entry

### Synopsis

This will generate an @article bibtex entry e.g.
	
@article{CitekeyArticle,
	author   = "<Lastname, FirstName>",
	title    = "<Title>",
	journal  = "<Journal>",
	year     = <2002>,
	volume   = "<50>",
	number   = "<1>",
	pages    = "<1--10>",
	month	 = <jul>,
	note	 = "<Note>"
}

Required:
	- author
	- title
	- journal
	- year
Optional:
	- volume
	- number
	- pages
	- month
	- note
<>: indicates that it is a example value and should be changed.	


```
bibcli article [flags]
```
