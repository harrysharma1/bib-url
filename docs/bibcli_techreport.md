## bibcli techreport

Generate @techreport bibtex entry

### Synopsis

This will generate an @techreport bibtex entry e.g.
	
@techreport{CitekeyTechreport,
	title       = "<Title>",
	author      = "<Lastname, Firstname>",
	institution = "<Institution>",
	address     = "<Address>",
	number      = "<50>",
	year        = <2002>,
	month       = <jul>
}

Required:
	- title
	- author
	- institution
	- address
	- number
	- year
	- month
Optional:
	- !*
<>: indicates that it is a example value and should be changed.
